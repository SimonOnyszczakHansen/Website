1. Introduction & Overview

Welcome to my Password Generator project! This application creates secure, memorable passwords by combining personal “interests” (or favorite words) with random elements. It supports both Danish and English language settings, automatically detecting your browser’s locale.

The interface is built with HTML, vanilla JavaScript, and Bootstrap for icons, plus a custom CSS theme for styling. Below is a quick look at the final UI:

Key highlights:

  • Add your own interests, favorite numbers, or colors.
  • Dynamically enforces minimum password length and includes special characters.
  • Language auto-detect (English or Danish).
  • A built-in strength meter checks for weak patterns.
  • Generates site-specific passwords by appending a portion of each website’s name.

2. How It Works

The core logic lives in script.js. As users add tags (e.g., “soccer,” “beer,” or “23”), these are merged into a base password with optional special-character transformations (e.g., converting “æ” or “ø” to ASCII). The app also capitalizes the first letter of each interest if desired and inserts user numbers at specific points.

Below is a simplified snippet from the password-generation logic, showing how a special character is inserted and interests are capitalized:

Once the base password is generated, you can append a subset of a site’s name (“Facebook,” “GitHub,” etc.) to get unique passwords per service.

3. Usage & Extra Features

After clicking “Generate Password,” you’ll see a base password plus site-specific passwords. Some noteworthy features:

  • Strength Meter: Checks uppercase, lowercase, numeric, and special characters, and flags any known weak patterns.
  • Video Tutorial Modal: A quick demo of adding tags, setting length, etc.
  • Predefined Sets: Offers recommended interests, colors, and numbers for faster input.
